# Steam Key Generator
Useless but fast key generator for Steam.

## How To Use
* Go to project folder
* Compile *main.c*
* `executable` `type` `number`
## Important notes
* `type` can be 1, 2 or 3
    * 1 - XXXXX-XXXXX-XXXXX
    * 2 - XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
    * 3 - XXXXXXXXXXXXXXX XX
* `number` is a natural number from 1 to 999999
